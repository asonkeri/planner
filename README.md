# Planner prototype

## About

This project is about trying to improve the design of a table with draggable items that I encountered in a work project and didn't think the implementation was very good.
The table is supposed to be a calendar or gantt-table like entity that can contain items spanning one or more days.
I was using this project to try to find better, cleaner design for especially the drag-and-drop functionality.
Also the aim was to practice my Typescript and React skills.

## Getting started
- Install dependencies: `yarn`
- Run the dev server: `yarn dev`
